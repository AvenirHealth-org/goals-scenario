# Output Indicators

Goals produces the following output indicators. Each is a multidimensional array stored in long format. The **Columns** column lists the columns for this indicator type. The `year` dimension is always present as the final column and
stores calendar years.

Indicator names are passed to `output_indicators` in the [config file](cli.md). You can also run `goals-scenario indicators` in the terminal to see this table.
